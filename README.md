# 💰 我的财务健康看板

一个基于 Streamlit 的个人记账数据处理 Web 应用，帮助您管理日常收支，分析财务状况。

## ✨ 功能特性

### 📊 核心功能
- **内置示例数据**：自动生成60条模拟流水数据（2026年6月），覆盖餐饮、交通、购物、房租/水电、娱乐、医疗等类别
- **数据录入与上传**：支持手动添加收支记录，支持CSV文件上传（覆盖或追加）
- **趋势分析**：使用 Plotly 绘制每日收支变化组合图和分类支出占比环形图
- **财务诊断与预警**：核心指标卡片、恩格尔系数计算、超支预警、超支日历
- **数据导出**：一键导出完整数据为CSV文件

### 🎨 视觉设计
- 蓝紫色到粉色的柔和渐变背景
- 毛玻璃效果卡片设计
- 数字和标题带有柔和阴影
- 按钮悬停缩放动画
- 所有交互带 Toast 提示反馈

## 📖 使用说明

### 数据录入与上传页
1. **手动添加记录**：选择日期、类别、输入金额和备注，点击添加
2. **上传CSV文件**：选择CSV文件，选择覆盖或追加模式，点击上传

### 趋势分析页
1. 使用月份选择器切换查看不同月份
2. 查看每日收支变化组合图（柱状图+折线图）
3. 查看分类支出占比环形图

### 财务诊断与预警页
1. 查看本月总收入、总支出、结余核心指标
2. 查看恩格尔系数及解读
3. 查看超支预警提示
4. 查看超支日历（红色标注超支日期）

### 数据导出
点击"导出当前完整数据为CSV"按钮下载数据文件

## ❓ 常见问题

**Q: 为什么看不到数据？**

A: 应用启动时会自动生成60条模拟数据。如果数据为空，请检查是否正确上传了CSV文件。

**Q: 如何添加自定义类别？**

A: 在 `project2.py` 中找到类别映射函数进行修改。

**Q: 数据会保存吗？**

A: 数据保存在 `st.session_state` 中，刷新页面后会重置。建议定期导出数据。
